# Goshala-application-
Application for modern goshala management. It combines cow tag scanning (QR/RFID/NFC) with finance and donation tracking, helping caretakers manage herd health and milk yield while donors receive receipts and transparent reports.


---
## ✨ Features
- Cow Tag Scanning (QR/RFID)
- Herd Management (profiles, health, milk yield, vaccination)
- Donation & Billing (payment gateway integration, receipts)
